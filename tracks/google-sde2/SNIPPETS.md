# Code Snippets (Interview-Ready)

This folder contains **copy/paste** implementations for the most common SDE-2 patterns.

Use them like this:
- Learn the **pattern + invariant** from `foundations/` and `patterns/`
- Then practice coding using these snippets as your “muscle memory” reference

Primary language: **Python** (more can be added if you want Java equivalents for all).

---

## Python snippet index

- `snippets/python/arrays.py` — prefix sums, intervals, Kadane, products
- `snippets/python/two_pointers_window.py` — two pointers, sliding window, min window, deque max
- `snippets/python/binary_search.py` — lower bound, rotated search, search-on-answer template
- `snippets/python/linked_list.py` — reverse, merge, cycle, remove-nth
- `snippets/python/stack_queue.py` — valid parentheses, monotonic stack, histogram
- `snippets/python/backtracking.py` — subsets, permutations, combination sum
- `snippets/python/trees.py` — traversals, LCA, validate BST, max path sum, serialize/deserialize
- `snippets/python/graphs.py` — BFS/DFS, topo (Kahn), islands, multi-source BFS, Dijkstra
- `snippets/python/dp.py` — house robber, coin change, LIS, LCS, edit distance, word break
- `snippets/python/union_find.py` — DSU + redundant connection template
- `snippets/python/bit.py` — xor tricks, count bits, hamming distance, max xor
- `snippets/python/greedy.py` — jump game II, gas station, intervals greedy
- `snippets/python/maths.py` — gcd/lcm, fast pow, sieve, trailing zeroes
- `snippets/python/sorting.py` — quickselect (kth largest)
