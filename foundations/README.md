## How the foundations notes are organized

Every major topic file aims to include:

1. **Concept overview** — When to use the technique.  
2. **Core algorithms** — Pseudocode or sketches and complexity.  
3. **Advanced / edge cases** — Variants and pitfalls.  
4. **Pattern recognition** — How to spot the pattern in a problem.  
5. **Code** — Python (and sometimes Java where noted).  
6. **Interview strategy** — How to approach under time pressure.  
7. **Quick revision** — Formulas and one-line triggers.  
8. **See also** — Cross-links to related topics and patterns.
