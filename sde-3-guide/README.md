# SDE-3 Prep Guide

High-level guide for **SDE-3 / Senior Software Engineer** coding and system-design preparation. SDE-3 interviews emphasize optimal solutions, edge cases, trade-offs, and often combine DSA with low-level design or concurrency.

## Main guide

**[sde-3-guide.md](sde-3-guide.md)** — Covers:

1. **Advanced DSA** — Combined topics (e.g. Segment Tree + sliding window, Trie + graph). Links to [Advanced DP](../patterns/dp-advanced.md), [Advanced Graphs](../advanced-dsa/advanced-graphs.md), [Tries & Segment Trees](../advanced-dsa/trie-segment-trees.md).
2. **Low-Level Design (LLD)** — Rate limiter, in-memory file system, LRU/LFU cache; extensible code and SOLID.
3. **Concurrency** — [Concurrency patterns](../concurrency/concurrency-patterns.md): reader-writer, producer-consumer, deadlock avoidance.
4. **Top 20 curated problems** — Word Search II, Alien Dictionary, Serialize/Deserialize Tree, LFU Cache, Merge K Lists, Median of Two Sorted Arrays, Sliding Window Maximum, Trapping Rain Water, Critical Connections, and more.
