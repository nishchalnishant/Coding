# Advanced DSA — SDE-3 Level

Advanced data structures and algorithms expected at Senior Engineer level.

## Contents

- **[Advanced Graphs](advanced-graphs.md)** — Tarjan's SCC, Bridges and Articulation Points, Union-Find (DSU) with path compression and union by rank. Pattern recognition and interview strategy.
- **[Tries and Segment Trees](trie-segment-trees.md)** — Trie (standard + binary for max XOR), Segment Tree (build/update/query, lazy propagation), Fenwick Tree. Pattern recognition and interview strategy.
- **[System Design Algorithms](system-design-algorithms.md)** — Rate limiting (token bucket, sliding window), consistent hashing, leader election, quorum. For coding rounds that touch system design.

Use these with the [SDE-3 Guide](../sde-3-guide/sde-3-guide.md) and [SDE3 DSA Roadmap](../SDE3_DSA_ROADMAP.md) for full preparation.
